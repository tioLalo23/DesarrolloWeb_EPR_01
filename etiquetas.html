Etiquetas de texto

Distribucion de texto

p

La etiqueta p delimita texto que se estructura como un párrafo. Es el más usado para texto plano. Se adapta al tamaño de la ventana. Al ser un párrafo, no admite saltos de línea (con Enter) ni etabulaciones. Por ejemplo:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Ut vel enim eu nisl mollis semper non ut nunc. Aliquam nisl dui, auctor eget malesuada at, dapibus sit amet mi. Pellentesque.

Para agregar un salto de linea dentro de una etiqueta p, se utiliza la etiqueta br:

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Ut vel enim eu nisl mollis semper non ut nunc.
Aliquam nisl dui, auctor eget malesuada at, dapibus sit amet mi.
Pellentesque.

hr

La etiqueta hr funciona para delimitar espacialmente las secciones del sitio. Coloca una linea horizontal divisoria.

pre

La etiqueta pre coloca texto con un formato tal y como está escrito en el código, así que es posible poner dobles espacios, tabuladores y saltos de línea con el teclado.

 Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
               Ut vel enim eu nisl mollis semper non ut nunc.               
                            Aliquam nisl dui, auctor eget malesuada at, dapibus sit amet mi. Pellentesque.

blockquote

La etiqueta blockquote ingresa texto con un formato de "cita". Lo enfatiza dentro del sitio con una sangría. Ideal para citas textuales largas.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vestibulum at tortor at nulla consequat laoreet vel non elit. Phasellus ullamcorper eget eros et porta. Donec laoreet nulla a neque feugiat, et faucibus turpis varius. Donec eget iaculis dolor. Duis congue semper tincidunt. Maecenas consectetur urna quis aliquet bibendum. Nullam feugiat fringilla nisl sit amet blandit. Praesent lectus quam, sollicitudin eu.

Formatos de texto

em

La etiqueta em se utiliza para enfatizar texto. Da formato similar a la letra cursiva.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

strong

La etiqueta strong tambien enfatiza texto. Da formato similar a la letra en negritas.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

small

La etiqueta small introduce texto pequeño. Puede ser usada para notas pequeñas o información legal.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

s

La etiqueta s muestra texto cancelado (tacha el texto). Puede usarse para indicar errores o correciones a un texto previo.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

cite

La etiqueta cite muestra texto procedente de fuentes externas, por ejemplo, titulos de obras.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

q

La etiqueta q representa cita textual. Coloca comillas y acomoda la cita adecuadamente.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

code

La etiqueta code muestra texto en formato de código, de manera que se diferencíe del texto normal.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.\n {color="blue"}

i

La etiqueta i coloca texto en cursivas (itálica). Puede sustituir a em, aunque también puede usarse dentro de la etiqueta p.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

b

La etiqueta b coloca texto en negritas (bold font). Puede susituir a strong, aunque también puede usarse dentro de la etiqueta p.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

u

La etiqueta u muestra texto subrayado.

Vestibulum sed dolor erat. Donec pulvinar diam scelerisque, tempus dui.

Disposición de contenido y elementos adicionales

Listas con viñetas

Una lista con viñetas puede ser creada con la etiqueta ul. Dentro, cada elemento de la lista será delimitado por etiquetas li

  • Lorem
  • ipsum
  • dolor
  • sit
  • amet

Listas numeradas

Una lista numerada puede ser creada con la etiqueta ol. Al igual que en ul, cada elemento de la lista será delimitado por etiquetas li

  1. Lorem
  2. ipsum
  3. dolor
  4. sit
  5. amet

Definiciones

Es posible crear glosarios (diccionarios) con de la etiqueta dl. Dentro de ella, cada término a definir se coloca entre etiquetas dt, mientras que la definición se coloca entre etiquetas dd

WWW
World Wide Web
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

a

La etiqueta a permite agregar hipervinculos al texto. Permite navegar a otro sitio o archivo al hacer clic en él. En enlace se coloca entre comillas como atributo href de la etiqueta.

Lorem ipsum dolor sit amet.

abbr

La etiqueta abbr permite mostrar mensajes flotantes en el texto al pasar el puntero del ratón por encima de él. El mensaje debe especificarse en el atributo title de la etiqueta

Lorem ipsum dolor sit amet.

Formatos dentro del párrafo

dfn y var

Dentro de un párrafo, es posible resaltar texto con la etiqueta dfn, así como con la etiqueta var. Pueden ser usados diferenciar conceptos (dfn) y variables (var) del resto del texto. A diferencia de la etiqueta i, no puede usarse fuera de un párrafo.

samp y kbd

Por otro lado, la etiqueta samp y la etiqueta kbd pueden ser usadas para enfatizar texto. El primero para texto de muestra y el segundo para diferenciar teclas o botones a presionar en teclado o pantalla.

sub y sup

Las etiquetas sub y sup pueden usarse para indicar subíndices y superíndices, especialmente en fórmulas. Ambas necesitan de apertura y cierre, encerrando el texto que irá en los índices. Por ejemplo, "La fórmula del dióxido de carbono se escribe como CO2", o bien "Este programa es patrocinado por Amazon MusicTM"

mark

Es posible resaltar texto usando la etiqueta mark, dando el efecto de un marcatextos. Es util para resaltar información de forma bastante llamativa, pero puede contrastar con el diseño si no se usa adecuadamente.
